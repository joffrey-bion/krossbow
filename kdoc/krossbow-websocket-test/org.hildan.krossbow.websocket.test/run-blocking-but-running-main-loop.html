fun runBlockingButRunningMainLoop(timeout: <ERROR CLASS>, block: suspend CoroutineScope.() -> Unit)
Content copied to clipboard

Runs a new coroutine and blocks the current thread interruptibly until its completion.

While the thread is blocked from the outside, this method still runs the iOS main loop periodically concurrently with the given block to make sure every queued operation (in NSOperationQueue.mainQueue) can progress.
